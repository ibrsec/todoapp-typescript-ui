<a name="quick-setup"></a>
## 🛫 Quick Setup

```sh
# clone the project
git clone https://github.com/ibrsec/todoapp-typescript-ui.git

# enter the project directory
cd todoapp-typescript-ui

# install dependency
npm install || yarn install

# develop
npm run dev || yarn start
```
